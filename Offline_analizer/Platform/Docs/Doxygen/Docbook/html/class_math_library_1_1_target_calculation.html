Calculates a rectangle around the target and based on this defines the middle of the side facing the ego (closest to the ego).

Also transforms the distances of all edges and sides of the target from the Rear Center Axle coordinate system to the Aligned Camera Coordinate System (ACCS)

Parameters
heightHeight of the target
widthWidth of the target
lengthLength of the target
heading_angleHeading angle of the target in relation to the ego
dxLongitudinal distance from the ego rear center axle to the target rear bumper
dyLateral distance from the ego rear center axle to the target rear bumper
offset_dx_fvMounting position in longitudinal direction of the camera in reference to the ego rear center axle
offset_dy_fvMounting position in lateral direction of the camera in reference to the ego rear center axle

Calculation functions

Calculate a rectangle around the target and therefor the distances of all 4 edges + middle of sides

Transform the coordinates from the ego rear center axle to the camera position

Step 1: Find the nearest side depending on the Sensor Coordinate System since this is what the real sensor would see

BUT the values of the distances are based on RCA!

Step 2: Define Reference Point

Implementation partly based on Cloe Code! https://sourcecode01.de.bosch.com/projects/PJDC/repos/pj-dc_int/browse/dpc_apl/tools/cloe/src/utility/radar_sensor_mockup.h

Fill in data of the reference point according to the defined Reference point

Sensor Coordinate System defines which side is reference, RCA clarifies the values.

Calculates a rectangle around the target and based on this defines the middle of the side facing the ego (closest to the ego).

Also transforms the distances of all edges and sides of the target from the Rear Center Axle coordinate system to the Sensor Position Coordinate System (Sensor_CS)

Parameters

widthWidth of the target
lengthLength of the target

dyLateral distance from the ego ref point to the target ref point
vxLongitudinal relative velocity of the target
vyLateral relative velocity of the target
mount_pos_dxMounting position in longitudinal direction of the sensor in reference to the ego rear center axle
mount_pos_dyMounting position in lateral direction of the sensor in reference to the ego rear center axle
mount_pos_yawMounting position in lateral direction of the sensor in reference to the ego rear center axle
ego_current_ref_pointDefines the reference point (center of coordinates) on the ego vehicle in which the data is given
tgt_current_ref_pointDefines the reference point on the target in which the data is given

Calculation functions. Inputs are expected to be:

Ego - Rear Center Axle

Target - Middle of the geometry

Read the dimensions of the target

The length must not be 0, otherwise it would be divided by 0 in further calculations

Calculate a rectangle around the target and therefor the distances of all 4 edges + middle of sides

Ego: Rear Center Axle

Calculate a rectangle around the target and therefor the distances of all 4 edges + middle of sides

Ego: Sensor Position

Calculate Reference Point

Step 1: Find the nearest side depending on the Sensor Coordinate System since this is what the real sensor would see

BUT the values of the distances are based on RCA!

Step 2: Define Reference Point
