# Contributing to Gen7_Autochecker

Thank you for your interest in contributing! Follow the steps below to push your first code to this repository.

## Prerequisites

- [Git](https://git-scm.com/downloads) installed on your machine
- A [GitHub account](https://github.com/join)

## How to Push Your First Code

### 1. Fork the Repository

Click the **Fork** button in the top-right corner of the repository page on GitHub. This creates a copy of the repository under your own GitHub account.

### 2. Clone Your Fork Locally

```bash
git clone https://github.com/<your-username>/Gen7_Autochecker.git
cd Gen7_Autochecker
```

### 3. Create a New Branch

Always create a new branch for your changes instead of working directly on `main`:

```bash
git checkout -b my-feature-branch
```

### 4. Make Your Changes

Add or edit the files you want to contribute.

### 5. Stage and Commit Your Changes

```bash
git add .
git commit -m "Add a short description of your change"
```

### 6. Push Your Branch to GitHub

```bash
git push origin my-feature-branch
```

### 7. Open a Pull Request

1. Go to your fork on GitHub.
2. Click **Compare & pull request**.
3. Fill in a title and description explaining your changes.
4. Click **Create pull request**.

A project maintainer will review your contribution and merge it when it's ready.

## Code Style

- Keep code clean and readable.
- Add comments where necessary to explain non-obvious logic.
- Make sure your changes do not break existing functionality.

## Questions

If you have any questions, feel free to open an [issue](https://github.com/tinschert/Gen7_Autochecker/issues) on GitHub.
